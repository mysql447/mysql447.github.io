## 第3章　從零開始，降低門檻學Python
### 10. 快速開始上手Python基本語法變數的第一個字母要用英文
- 變數
  - 在a=1的式子中，a就是變數，之後只要輸入a就會代入1。
  - 在命名變數時，需要注意，**變數的第一個字母要用英文**，之後可以搭配數字，例如：a2。若以2a 命名變數，就會無法使用。
- 3種常用的字串使用方式
  - 使用加號「+」串接不同的文字資料。
  ```
  a="Hello"
  b="Orli"
  print(a+b)
  
  結果
  HelloOrli
  ```
  - 使用乘號「*n」重複文字資料。
  ```
  a="Hello"
  print(a*2)
  
  結果
  HelloHello
  ```
  - 使用「in」查詢文字裡面是否包含某些字符。
  ```
  a="我是Orli"
  print("Orli" in a)
  print("你好" in a)
  
  結果
  True     #變數a包含Orli
  False    #變數a不包含你好
  ```
- 不同的資料型態無法相加。
  - 若變數a是文字，變數b是數字，在沒有進行資料型態轉換下做運算，程式會出現錯誤訊息。
  - 因此，若要相加兩者，要將b轉為字串。
  ```
  a="這瓶飲料售價是"
  b=25
  print(a+b)
  print(a+str(b))
  
  結果
  TypeError
  這瓶飲料售價是25
  ```

- 基本運算式
  - 取商數「//」
  - 取餘數「%」
  - 次方「**」
- 關係運算式
  - 數值間的大小關係，運算結果可用布林值「True」或「False」呈現。
  - 使用「=」，在Python 中代表「設立變數」。如：a=1，變數a為1
  - 使用「==」，在Python 中代表「等於」。如：1+1==2，1加1等於2為數學運算
  - 使用「！=」，在Python 中代表「不等於」。
- 邏輯運算式
  - and和、且
  - not否、非
  - or或

- 條件判斷式
  - 透過if、else執行
  - 多重條件要使用elif

- 串列list
  - 可以儲放多筆資料，如：文字、數字、另一個串列資料，以序列方式呈現。
  - 使用中括號［］表示。
  - 變數名稱=[數字，"文字"，［"文字"，數字］，"文字"］
  - 以位置順序辨別資料。第一個位置是從0開始計數。
- 串列list使用方式
  - 查詢或取得串列中第n個位置的資料。位置的數值要使用［］。
  ```
  a=["apple",22.5,"飲料",50,2025]
  print(a[0])     #資料位置為0
   
  結果
  apple
  ```
  - 取得串列位置0到位置2**前**的內容，也就是位置0、1的資料。
  ```
  a=["apple",22.5,"飲料",50,2025]
  print(a[0:2])
   
  結果
  ["apple",22.5]
  ```
  - 以「del」刪除串列位置0到位置2**前**內容，也就是刪除位置0、1的資料。
  ```
  a=["apple",22.5,"飲料",50,2025]
  del a[0:2]
  print(a)
   
  結果
  ["飲料",50,2025]
  ```  
  - 用「**index**」查詢資料在串列中為第幾個**位置**。
  ```
  a=["apple",22.5,"飲料",50,2025]
  print(a.index(22.5))
   
  結果
  1
  ```  
  - 用「**len**」查詢串列中的**資料數量**。
  ```
  a=["apple",22.5,"飲料",50,2025]
  print(len(a))
  print(len(["apple",22.5]))    #[]內共有幾筆
   
  結果
  5
  2
  ```  
  - **最小值min**或**最大值max**（僅限數值資料）。
  ```
  a=[666,22.5,15,50,2025]
  print(min(a))
  print(max(a))
   
  結果
  15
  2025
  ```
  - 用「**count**」計算資料在串列中**出現的次數**。
  ```
  a=["apple",22.5,"飲料",50,2025]
  print(a.count(apple))
     
  結果
  1
  ```
  - 將「**單個**」資料新增到串列**後方**：**append**。
  ```
  a=["apple",22.5,"飲料",50,2025]
  a.append("Hello")
  print(a)
     
  結果
  ["apple",22.5,"飲料",50,2025,"Hello"]
  ```
  - 將「**多個**」資料新增到串列**後方**：**extend**或「**+**」。
  ```
  a=["apple",22.5,"飲料",50,2025]
  a.extend(["紅茶",30])
  print(a)
  b=["紅茶",70]
  c=b+a+["Hello"]
  print(c)
     
  結果
  ["apple",22.5,"飲料",50,2025,"紅茶",30]
  ["紅茶",70,"apple",22.5,"飲料",50,2025,"紅茶",30,"Hello"]
  ```
  - 將資料**新增到指定的位置**：**insert**。
  ```
  a=["apple",22.5,"飲料",50,2025]
  a.insert(0,"Hello")    #將資料「Hello」新增到位置0
  print(a)
     
  結果
  ["Hello","apple",22.5,"飲料",50,2025]
  ```
  - 將資料以**pop 刪除**。
    - pop 內沒填數值，預設刪除**最後一筆資料**，回傳被刪除的資料。
    - 被刪除的資料不會出現在變數中
  ```
  a=["apple",22.5,"飲料",50,2025]
  print(a.pop())   #預設刪除最後一筆
  print(a.pop(0))
  print(a)
     
  結果
  2025
  apple
  [22.5,"飲料",50]
  ```
  - 用 「**reverse** 」將資料順序**反轉**。
  ```
  a=["apple",22.5,"飲料",50,2025]
  a.reverse()
  print(a)
     
  結果
  [2025,50,"飲料",22.5,"apple"]
  ```
  - 用 「**sort**」將數值資料**從小到大排列**（僅限數值資料）。
  ```
  a=[666,22.5,15,50,2025]
  a.sort()
  print(a)
   
  結果
  [15,22.5,50,666,2025]
  ```
- 字典dictionary(dict)
  - 可以儲放帶有**索引值**的資料，使用大括號｛｝和冒號：表示。
  - 由一個關鍵字key（字），回傳對應的值 value（對於字的解釋或內容）給使用者。
  - key 只能是單一的文字（str）或數值（int、float），但value 資料型態不限。
  - 變數名稱=｛key：value，key：value｝
  ```
  #表示方式一
  a={"apple":30,"飲料":50,"紅茶":45}
  
  #表示方式二
  a={}
  a["apple"]=30
  a["飲料"]=50
  a["紅茶"]=45
  ```
- 字典dictionary使用方式
  - 查詢關鍵字或對應值。
  ```
  a={"apple":30,"飲料":50,"紅茶":45}
  print(a["飲料"])

  結果
  50
  ```
  - 用 「**keys()**」查詢**所有關鍵字**。
  ```
  a={"apple":30,"飲料":50,"紅茶":45}
  print(a.keys())

  結果
  dict_keys(["apple","飲料","紅茶"])
  ```
  - 用 「**values()**」查詢**所有對應值**。
  ```
  a={"apple":30,"飲料":50,"紅茶":45}
  print(a.values())

  結果
  dict_values([30,50,45])
  ```
  - 用 「**len**」查詢 dictionary **資料數量**。
    - key和value算一組
  ```
  a={"apple":30,"飲料":50,"紅茶":45}
  print(len(a))

  結果
  3
  ```
  - 用 「**clear()**」**清空** dictionary 資料。
  ```
  a={"apple":30,"飲料":50,"紅茶":45}
  a.clear()
  print(a)

  結果
  {}
  ```
  - 取得 dictionary 中最小值min或最大值max。
  ```
  a={"apple":30,"飲料":50,"紅茶":45}
  print(min(a))
  print(max(a))

  結果
  apple
  飲料
  ```
